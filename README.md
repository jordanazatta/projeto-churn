# 📊 Análise de Churn de Clientes

Aplicação desenvolvida em **Python** e **Streamlit** para análise de churn (cancelamento de clientes) de uma empresa de telecomunicações.

O projeto permite visualizar indicadores, consultar dados, explorar gráficos estáticos e interativos, gerar relatórios e compartilhar informações por e-mail.

---

## 🌐 Acesse a aplicação

**Link do projeto:**

https://jordana-zatta.streamlit.app/

---

## 📌 Funcionalidades

- Panorama com os principais indicadores
- Consulta de tabelas
- Gráficos estáticos
- Gráficos interativos
- Relatório gerencial
- Envio de gráficos por e-mail
- Envio do relatório por e-mail

---

## 📈 Indicadores apresentados

- Total de clientes
- Clientes cancelados
- Clientes ativos
- Taxa de churn
- Categorias de cancelamento
- Principais motivos de cancelamento
- Taxa de churn por tipo de contrato

---

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib
- SMTP (envio de e-mails)
